Description:
The dataset used in this project is sourced from Kaggle by searching for Amazon product reviews : amazon_reviews.csv.

This project performs sentiment analysis to classify customer reviews as Positive, Negative, or Neutral.
It also detects basic emotions such as Joy and Anger using NLP techniques.
The analysis includes data cleaning, preprocessing, sentiment and emotion classification, and visualization
of distributions to understand customer opinions and trends.

Insights:
1. Majority of the reviews are Positive, indicating high customer satisfaction.
2. Negative reviews are fewer and mainly highlight delivery or product issues.
3. Joy is the dominant emotion among expressive reviews, reflecting good product acceptance.
4. Very few reviews indicate Anger, suggesting minimal strong dissatisfaction.
5. Sentiment patterns provide insights useful for marketing, product development, and business decisions.

Output:
Sentiment and emotion counts for all reviews.
Visualizations of Sentiment Distribution and Emotion Distribution using charts.
Analysis results are available in the GitHub repository.
